    746. Meta Command Notes:
    - The <"page size id"> meta value captures paper size, e.g. A4, B4, Letter, Custom, etc...

    For custom page size use "Custom"

    - The SUBMODEL metas below enable font and background settings for nested submodels and callouts.
    - The SUBMODEL_FONT meta is supported for up to four levels.
    - The SUBMODEL_FONT_COLOR meta is supported for up to four levels.
    - The SUBMODEL_BACKGROUND_COLOR meta is supported for up to four levels.

    Four level colours #FFFFFF, #FFFFCC, #FFCCCC, and #CCCCFF are predefined.

    - The meta value captures the 7 stud style types.
    0 None
    1 Thin line logo
    2 Outline logo
    3 Sharp top logo
    4 Rounded top logo
    5 Flattened logo
    6 High contrast without logo
    7 High contrast with logo

    - The meta value captures the 4 annotation icon style types.
    0 None
    1 Circle
    2 Square
    3 Rectangle
    4 LEGO element

    - The meta value captures the 5 border line types.
    0 None
    1 Solid ----
    2 Dash - -
    3 Dot ....
    4 Dash dot -.-.
    5 Dash dot dot -..-

    - The <"font attributes"> meta value is a comma-delimited <"string"> of 10 attributes.
    Example font attributes <" Arial, 64, -1, 255, 75, 0, 0, 0, 0, 0 ">
    1 FamilyName - "Arial"
    2 PointSizeF - 64 size of font, -1 if using PixelSize
    3 PixelSize - -1 size of font, -1 if using PointSizeF
    4 StyleHint - 255 = no style hint set, 5 = any style, 4 = system font, 0 = Helvetica, etc...
    5 Weight - 75 = bold, 50 = normal, etc...
    6 Underline - 0 = disabled, 1 = enabled
    7 Strikeout - 0 = disabled, 1 = enabled
    8 StrikeOut - 0 = disabled, 1 = enabled
    9 FixedPitch - 0 = disabled, 1 = enabled
    10 RawMode - 0 obsolete, use default value
